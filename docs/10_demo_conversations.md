# 10. 대화 예시 3개 (PlayMCP 등록용)

> PlayMCP 임시등록 → "대화 예시" 칸에 아래 **사용자 발화**를 그대로 입력(최대 3개).
> 전부 **무키·항상 완결**(명식·해석은 내부 계산, 날씨는 best-effort라 없어도 정상)이라 데모가 중간에 끊기지 않는다. 칩으로 이어지는 여정을 보여주도록 구성.

---

## 예시 1 — 명식 + 오늘의 기운 (데일리 훅)
> **1995년 3월 2일 밤 8시에 태어났어요. 제 사주 명식이랑 오늘의 기운 알려주세요.**

- 흐름: `computeSajuChart`(명식+프로필 코드) → 칩 `오늘의 기운` → `getTodayFortune`(점수·키워드·럭키 색/숫자/음식/시간대·do&don't, 위치 주면 날씨)
- 보여주는 것: 정확한 만세력 + 매일 바뀌는 데일리 처방 + "매일 아침 받기" 넛지.

## 예시 2 — 궁합 카드 (공유·바이럴)
> **저는 1992년 11월 3일생, 친구는 1990년 5월 15일생이에요. 우리 궁합 봐주세요.**

- 흐름: `getCompatibility`(두 명식 → 점수·하트·캐치프레이즈·강점/주의 카드 + 공유 카드)
- 보여주는 것: 복사해 친구에게 보내는 공유 카드 + "나도 보기" 수신자 유도(바이럴 루프).

## 예시 3 — 사주 유형 + 택일 (시그니처 + 실용)
> **제 사주 유형이 뭔지 궁금해요. 그리고 올해 이사하기 좋은 날도 찾아주세요. (1988년 7월 21일 오전 9시생)**

- 흐름: `analyzePersonality`(사주 16유형·MBTI식: 코드/별명/연애·일·조언) → 칩 → `findAuspiciousDate`(이사 길일 Top3·손 없는 날)
- 보여주는 것: 공유형 "사주 유형" + 경쟁자에 없는 택일(개인 명식 기준 길일).

### 대표 이미지(등록 시)
- **정적 이미지**(움짤 금지), 서비스 성격 부합·고품질. 예: "🔮 사주 컨시어지 — 오늘의 기운·궁합·사주 유형" 텍스트가 들어간 단정한 카드형 이미지.
- 본선 Kakao Tools Widget 시안(채팅에서 제시한 카드 디자인)을 이미지로 다듬어 사용 가능.
