## 1. 서비스 소개 (복붙용)

**한 줄**: 생년월일시만 넣으면 만세력으로 사주를 계산해 **오늘의 기운·궁합·사주 유형·올해 운세·이름풀이·택일**을 재미있게 풀어주는, **사주로 오늘을 운영하는 데일리 비서**.

**짧은 소개(2~3문장)**: Saju Concierge(사주 컨시어지)는 검증된 만세력 엔진으로 사주 명식을 계산하고, 자체 작성한 해석 데이터로 **매일 바뀌는 데일리 럭키 키트·MBTI식 사주 16유형·공유용 궁합 카드·개인 맞춤 택일**을 제공합니다. 결과마다 다음 행동 칩과 공유 카드가 붙어 **버튼 누르듯 이어가고, 친구에게 공유**할 수 있습니다. 무저장·무로그인(stateless)·개인정보 6종 미수집.

## 2. 핵심 차별점 (경쟁자 대비)

PlayMCP에 이미 사주 MCP가 있음(1FATE '정통 명리학'=딥 명리, '사주봄'=쉬운 사주). 우리는 정반대 포지션:
- **재미·공유·매일 습관·대중 접근성** — 정통/전문이 아니라 "오늘을 운영하게 해주는 데일리 비서".
- **데일리 럭키 키트**(점수+키워드+럭키 색/숫자/방향/음식/시간대+do&don't, **매일 변주**) + **종합평**(점수→행동) + **나와 잘 맞는 인연**(띠·기운·행동).
- **공유 카드 + "나도 보기" 바이럴 훅** → 공유받은 사람이 바로 따라 씀(공개투표 모수↑).
- **택일**(개인 명식 기준 길일) = 경쟁자 0.
- **칩 여정**(조합 LLM 지시문으로 카카오 AI채팅에서도 칩 생존 — 라이브 검증).

## 5. 서비스 화면 캡처 가이드 (⚠️ 비즈 심사 = 별개 게이트)

kpass(1번째 작품)는 **사업자 정보 미검증 + 빈약한 화면**으로 비즈 심사 **2회 반려**됨 → 화면을 충실히 준비.

**캡처 방법(가장 확실)**: `playmcp.kakao.com` → **AI 채팅** → 도구함에서 **사주 컨시어지만 켜고**(다른 MCP는 끄면 깔끔) → 대화 예시 발화 입력 → 렌더된 카드를 캡처.
- 캡처 ①: **오늘의 기운 카드**(점수·키워드·종합평·럭키·인연) — 서비스 핵심 가치가 한눈에.
- 캡처 ②: **칩 노출 화면**("💬 다음으로 물어보세요" + 공유 카드) — 이어가기/공유 UX.
- 캡처 ③(선택): **궁합 카드** 또는 **도구함에 사주 컨시어지 활성(Tools 7)** 화면.
> 본 세션(2026-06-30)에 플레이그라운드에서 위 화면들을 실제로 띄워 확인함(카드·칩·공유 정상 렌더).
